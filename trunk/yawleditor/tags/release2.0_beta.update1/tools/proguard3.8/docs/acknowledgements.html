Acknowledgements

ProGuard grew out of RetroGuard, which its author Mark Welsh has kindly made available under the GNU Lesser General Public License. RetroGuard is a very nice piece of code, but it only performed obfuscation. I started from the class file parsing code and wrote my own shrinker, optimizer, and obfuscator. At this point, both programs have little code in common.

Dirk Schnelle has contributed and maintained the first versions of the Ant task. I have rewritten the implementation for version 3.0, but the XML schema is still based on his work.
