Support Chiba! Requirements
  • JDK/JRE 1.4 or higher
  • a Servlet 2.2 compatible webcontainer
The current release was developed and tested with JDK 1.4.2 and Tomcat 4.1.30. When you're using these components installation is easy.
Installation (Tomcat 4.1/5.0)
  • download chiba-0.9.9.war
  • copy the WAR-file to $CATALINA_HOME/webapps
  • Important: Extract WEB-INF/lib/dom3-xercesImpl.jar and WEB-INF/lib/dom3-xml-apis.jar from the WAR-file. Move these libs to $CATALINA_HOME/common/endorsed. Make sure to remove/backup any other xercesImpl.jar and xml-apis.jar possibly residing there. Alternatively you might install the extracted libs as extensions in your $JAVA_HOME.
  • the step above is not necessary for Tomcat's LE-jdk14 editions
  • start Tomcat
  • point your browser to http://localhost:8080/chiba-0.9.9 to view the sample forms
In case nothing happens we would be happy if you give us a note with your system configuration and symptom description.
