STARTING YAWL - Worflow System (Engine Beta 8.2 + Editor 1.5)

To start YAWL please follow the following steps:

  • 1. If you installed YAWL with database support, click "Start Database" from the YAWL menu folder,
  • 2. Click "Start YAWL Service" from the start menu,
  • 3. Click "Administrate YAWL" from the start menu.

Please note that the YAWL Service may take a while to start.
