Dear Chiba user,
Thank you for your interest in Chiba. As you might know by now Chiba provides a free, standard-conformant Open Source XForms processor for everybody to use in their own applications. We're strongly believe that XForms has the potential to open new levels of productivity for data capturing and form-processing applications. We also believe that there's a strong need for an Open Source implementation to help spreading this technology and for security sensitive applications. Two developers are working full-time on Chiba to push it forward to a fully-conformant, production-quality implementation. Nearly three years lifetime, a growing community and over 10.000 downloads prove that we're taking this job seriously.
If you're already using or think of using Chiba please consider helping us to succeed by making a donation. We'll use this money directly to improve the code and support the community.
Thank you very much.
