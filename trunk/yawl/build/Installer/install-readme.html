STARTING YAWL

To start YAWL Please follow the following steps

  • If you installed YAWL with Database support, click start-database from the YAWL start menu folder
  • click start yawl service from the start menu
  • click administrate yawl from the start menu

If the administrate yawl link does not work straight away, this may be because yawl takes a minute to start. Please wait and try again.
